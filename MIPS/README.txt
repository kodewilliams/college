ABOUT THE PROGRAM
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - -

Execution Instructions:
- The program is run by loading the *.s file into QtSpim and subsequently pressing the green triangle (play button).
- It will prompt for the first integer, then the operator and lastly, the second integer for the operation.
- It will then try to do the operation given valid parameters, however if the operator is invalid, an error message will be thrown.
- If it found the answer successfully, it will print your operation and the answer.

Known Problems:
- The multiplication being done in this code is 32 bit instead of 64 bit.
